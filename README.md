# Digital Smart Attendance Face Recognition

## Overview
This project is a smart attendance system that uses face recognition to identify students or employees automatically.

## Features
- Face detection and recognition
- Automatic attendance marking
- Real-time camera input
- Attendance record storage

## Technologies Used
- Python
- OpenCV
- Face Recognition Library
- NumPy
- Pandas

## How to Run
1. Clone the repository
2. Install required packages
3. Run the main Python file

## Future Improvements
- Web dashboard
- Database integration
- Better accuracy in low light
